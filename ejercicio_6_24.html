Ejercicio 24 -> Obteniendo HTML remoto con AJAX Ir a Página Principal

En el siguiente código html realizaremos peticiones AJAX:

Objetos con peticiones AJAX:

Codigo Javascript:

$(document).ready(function () {
  $('#example').on('click', 'button.switch', function(){
    $.ajax('result.html', {
      beforeSend: function(){
      $('#status').text('Cargando...');
      }
    })
    .done(function(response){
      $('#result').html(response);
    })
    .always(function(){
      $('#status').text('Completado')
    })
  });
});
