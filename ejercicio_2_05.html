2.- Atravesando el DOM

Ejercicio 05 -> find() Ir a Página Principal

En el siguiente código html Atravesaremos el DOM utilizando la función find():

Listas con atributos:

Animals
Mammals
Dogs
Cats
Forest
Prairie
Reptiles
Snake
Lizard
Jungle

Codigo Javascript:

$(document).ready(function() {
//CSS selectors
          var result = $('#animals .creature');
          console.log(result);      
//DOM traversing (more efficient)
          result = $('#animals').find('.creature');
          console.log(result);
});

Codigo CSS:

            margin: 20px;
