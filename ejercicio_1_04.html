1.- Selección del DOM

Ejercicio 04 -> Selección por Atributos Ir a Página Principal

En el siguiente código html realizaremos selecciones por Atributo:

Formulario:

Codigo Javascript:

$(document).ready(function () {
  $('#prueba1 > div > input[required]').addClass('highlighted');
});

Codigo CSS:

.highlighted { color: white; background-color: blue;  }

Formulario:

Codigo Javascript:

$(document).ready(function () {
  $('#prueba2 > div > input[placeholder="Phone"]').addClass('highlighted');
});

Codigo CSS:

.highlighted { color: white; background-color: blue;  }

Formulario:

Codigo Javascript:

$(document).ready(function () {
  $('#prueba3 > div > input[placeholder*="Name"]').addClass('highlighted');
});

Codigo CSS:

.highlighted { color: white; background-color: blue;  }
