Ejercicio 28 -> Formulario con Checkbox Ir a Página Principal

En el siguiente código html realizaremos peticiones AJAX:

Objetos con peticiones AJAX:

Codigo Javascript:

$(document).ready(function () {
  $('#hasPets').on('change', function () {
    if ($(this).is(':checked')) {
      $('#pets-row').show();
    }
    else {
      $('#pets-row').hide();
    }
  });
  $('#hasPets').trigger('change');
});
